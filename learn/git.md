# GIT

## Создание репозитория

```git init```

## Добавление файла в слежку
```git add <git.md>```
```git add .```

## Добавление данных о себе в систему git
```git config --global user.email "andrey@example.com"```
```git config --global user.name "Andrey"```

## Коммит (сохранение)
```git commit -m "<commit name>"```
